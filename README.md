# Executing requirements
```
Composer must be installed 
PHP 5.5 or newer must be installed 
```
# Executing guide
* Donwload project
```
git clone https://github.com/jpbaterabsb/cp-php.git
```
* Execute composer install
```
composer install
```

* Execute composer start
```
composer start
```
